# 📊 Sistema de Avaliação SAP & DISC - Jorge

Este projeto foi desenvolvido para a gestão e análise de competências socioemocionais (SAP) e perfil comportamental (DISC) de jovens em formação.

## 🚀 Funcionalidades
- **Formulário Interativo:** Coleta de dados sobre as 7 competências SAP e as 16 tendências DISC.
- **Painel de Indicadores:** Visualização em tempo real das médias da turma (Gráfico de Barras).
- **Análise Individual:** Gráfico Radar para cada aluno, facilitando feedbacks personalizados.
- **Gestão de Dados:** Funções para imprimir laudos e zerar a base de dados para novas turmas.

## 🛠️ Estrutura do Projeto
- `index.html`: Interface de preenchimento para os alunos.
- `resultados.html`: Dashboard administrativo e análise de dados.

## 🎨 Identidade Visual
O projeto utiliza uma interface moderna em **Dark Mode**, com destaques em Laranja (`#ff5c3a`) e Ciano (`#00d2d3`), garantindo uma leitura clara e dinâmica.
